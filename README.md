# The Cozy Corner Cafe Menu

A beautiful, responsive cafe menu website built with HTML and CSS.

## Features

- Clean, modern design with warm coffee-shop colors
- Responsive layout that works on all devices
- Smooth hover animations on menu items
- Sticky navigation bar
- Dietary tags (Vegan, Vegetarian, Gluten Free)
- Popular item highlights

## Menu Sections

- **Coffee & Espresso** - Classic lattes, cappuccinos, cold brew
- **Tea & Non-Coffee** - Matcha, chai, herbal teas
- **Pastries & Baked Goods** - Croissants, muffins, brownies
- **Brunch** - Eggs benedict, French toast, granola bowls
- **Daily Specials** - Seasonal and limited-time offerings

## Usage

Simply open `index.html` in any web browser to view the menu.
